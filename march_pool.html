Project Overview

As the organizer of a March Madness bracket pool for the past eight years, I found it challenging to manage and update scores using a Google Sheet, which was also inconvenient for participants to track their scores. To streamline this process, I developed BracketBuddies, a live website using SQL and HTML, making score updates easier for me and score viewing more accessible for competitors.

Problem Statement:

Managing and updating scores for a March Madness bracket pool using a Google Sheet has been cumbersome for both the organizer and participants. The need for a more efficient and user-friendly solution led to the creation of BracketBuddies, a platform designed to simplify score tracking and updates.

Solution:

BracketBuddies is hosted on Google Firebase and connected to a PostgreSQL database, enabling live updates and ease of use. This setup allows for real-time score updates and provides participants with a seamless experience when viewing their scores

Skills Learned:

  • Firebase: Hosting and managing the website
  • PostgreSQL: Database management and real-time data updates
  • HTML: Building the user interface for the platform
